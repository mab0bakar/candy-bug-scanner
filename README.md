# 🍬 Candy Bug Scanner

An AI-powered code bug detector and auto-fixer that supports **any programming language**.
Paste your code, and the AI instantly finds every bug and returns a fully fixed version.

## ✨ Features
- 🔍 Auto-detects programming language (Python, JavaScript, Java, C++, Go, Rust, PHP, and more)
- 💥 Classifies bugs as Critical / Warning / Suggestion
- 🔧 Auto-fixes every issue and shows the corrected code
- ⬇️ Download the fixed code instantly
- 🎨 Beautiful candy-themed dark UI
